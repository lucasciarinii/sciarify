# Sciarify 🎵

**High school experiment** — a Spotify-inspired music player built from scratch in C# and WPF to practice desktop UI development and component logic.

## Quick Overview
- Spotify-inspired desktop music player
- Built with ![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=csharp&logoColor=white) ![WPF](https://img.shields.io/badge/WPF-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)

## Features
- 🎵 Music player with playback controls and progress slider
- ❤️ Mark tracks as favourites
- 📂 Playlist selection and management

## Why?
Built during high school to experiment with WPF layouts, controls and event-driven logic by cloning a real-world interface like Spotify. 🚀
